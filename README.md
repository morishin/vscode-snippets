# vscode-snippets
My snippets

## Install
macOS

```sh
ln -s $PWD/*.json ~/Library/Application\ Support/Code/User/snippets/
```

or

```sh
cp $PWD/*.json ~/Library/Application\ Support/Code/User/snippets/
```
